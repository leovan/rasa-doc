# 兼容性矩阵

在选择要[部署](deploy/deploy-rasa-pro-services.md)的 Rasa Pro Services 版本时，可以参考下表以查看哪个版本与安装的 Rasa Plus 兼容。

Rasa Pro Services 版本独立于 Rasa Plus 版本，只是它们共享相同的主版本号。

| Rasa Pro Services | Rasa Plus           |
| :---------------- | :------------------ |
| 3.0.x             | 3.3.x，3.4.x，3.5.x |
| 3.1.x             | 3.6.x               |
